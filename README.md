Approach (CRISP-DM)


Business Understanding – Defined the problem: which forecasting method most reliably predicts future values for this dataset?
Data Understanding – Explored the dataset to understand trends, seasonality, and historical patterns.
Data Preparation – Cleaned and structured the data for forecasting in Excel.
Modeling/Analysis – Applied multiple forecasting techniques and compared their predicted values against actual outcomes.
Evaluation – Measured forecast accuracy across methods to identify which approach minimized error most consistently.
Deployment – Visualized forecast vs. actual results in Excel to clearly show which method performed best and where each one struggled.


Key Insight

Identified which forecasting approach produced the most reliable predictions for this dataset, and pinpointed where each method's accuracy broke down — useful context for choosing the right technique depending on the data's behavior.

Why This Project

Applied the same structured CRISP-DM process used across my other projects to a forecasting tutorial dataset — moving past just generating a forecast, to actually evaluating which method deserves trust.
